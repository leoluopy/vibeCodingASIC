# sim — 仿真系统

## 内容

| 子目录 | 说明 |
|---|---|
| `cmodel/` | C/C++ 精度模型（与 RTL 对标） |
| `perf_model/` | 性能模型（周期精确 / 事件驱动） |
| `tests/` | 仿真测试用例与 golden data |
| `qemu/` | Qemu 虚拟机配置与 BSP |

## 职责

- 行业典型模型建模，确保 cmodel 精度与 RTL 一致
- 构建性能仿真模型，提前评估架构性能
- 软硬件需求拆分：明确哪些由硬件加速、哪些由软件完成

## 里程碑对应

- M1: 输出仿真系统开发清单
- M2: cmodel + perf_model 并行开发
- M3: Qemu 模型典型模型软件栈走通 + 仿真性能读出
