# VibeCoding RISC-V AI 芯片加速卡

## 项目概述

基于 **VibeCoding** 方法论，全流程打造一款 RISC-V AI 芯片加速卡。架构设计与系统调试由开发者主导，硬件及软件的基本编码与测试全部通过 VibeCoding 完成。

## 核心目标

1. **可行性探索** — 验证 VibeCoding 在 AI 芯片设计全流程中的可行性，探索人机协作的芯片研发新模式。
2. **开源共建** — 工作方法论、踩坑记录、完整仓库代码全部开源回馈社区。**流片成功后，每位开源贡献者将获赠一枚样片作为纪念。**

## VibeCoding 角色分工

| 拆分模块 | 任务 |
|------|------|
| **功能性能仿真建模** | 行业典型模型建模、软硬件需求拆分、cmodel 精度校准、性能仿真 |
| **硬件开发 / 时序验证** | RTL 开发、IC 验证、物理设计（时序收敛） |
| **软件开发 / 性能验证** | AI 编译器全栈编译优化、软件生态适配、功能及性能验证 |

## 目录结构

| 目录 | 说明 |
|------|------|
| `docs/` | 设计文档、需求规格、方法论与踩坑记录 |
| `hw/` | 硬件设计：RTL、IC 验证环境、物理设计脚本 |
| `sw/` | 软件栈：AI 编译器、Runtime、驱动、生态适配 |
| `sim/` | 仿真系统：cmodel、性能模型、精度验证 |
| `scripts/` | 工具脚本：构建、自动化测试、CI/CD |
