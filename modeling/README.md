# AI Chip Modeling

本目录专注于 AI 芯片的系统建模与性能分析，主要涵盖以下内容：

## 目标

- 对行业内常见 AI 大模型（Transformer、MoE、Diffusion 等）结构进行拆解与分析
- 基于模型结构推导关键硬件规格需求（算力、带宽、存储、互联拓扑等）
- 建立初步仿真建模，评估 AI 芯片在不同 workload 下的性能天花板
- 为芯片微架构设计提供量化依据与性能预估

## 目录结构规划

```
modeling/
├── README.md            # 本文件
├── models/              # 大模型结构拆解与分析
├── specs/               # 硬件规格需求推导
├── simulator/           # 性能仿真建模代码
└── results/             # 仿真结果与分析报告
```

## 工作流程

1. **模型分析** — 解构典型 AI 模型的算子组成、计算图特征、显存占用
2. **规格推导** — 根据模型需求推导算力（TOPS）、带宽（GB/s）、On-Chip SRAM 等指标
3. **仿真建模** — 基于 roofline model 或 cycle-accurate simulator 进行性能评估
4. **天花板分析** — 在给定硬件约束下估算吞吐量、延迟、利用率等关键性能指标
